# GraphQL Demo #

> Demo app using GraphQL and Apollo.

### Run App ###
```
yarn run installer
yarn build
yarn start
```

### Run Separate Apps ###
###### Server
```
cd server
yarn install
yarn start
```

###### Client
```
cd client
yarn install
yarn start
```

### Sites ###
* [UI Frontend](https://graphql-app.herokuapp.com)
* [GraphQL Server](https://graphql-app.herokuapp.com/graphiql)

### Application Stack ###
###### Server
* Node w/ Express
* GraphQL

###### Client
* Create React App
* GraphQL w/ Apollo
* Material UI
