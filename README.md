# GraphQL Demo #

> Demo app using GraphQL and Apollo.

### Run UI ###
```
yarn
yarn start
```

### Run API ###
```
yarn start:api
```

### Sites ###
* [UI Frontend](https://graphql-demo-mu.vercel.app)
* [GraphQL Server](https://graphql-demo-mu.vercel.app/api/graphiql)

### Application Stack ###
###### Server
* Node w/ Express
* GraphQL

###### Client
* Create React App
* GraphQL w/ Apollo
* Material UI
